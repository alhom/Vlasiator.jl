Vlasiator.jl

Data processing and analyzing tool for the numerical model for collisionless ion-kinetic plasma physics Vlasiator. This lightweight package is built upon its sister in Python Analysator, and is carefully designed for performance and ease of use. It becomes even more powerful when using together with other fantastic packages: combined with external packages like FieldTracer.jl and TestParticle.jl, it is possible to do all kinds of in-depth analysis.

Vlasiator.jl contains the following features:

  • Reading VLSV format data.
  • Extracting quantities from the simulation at a given point/line/cut.
  • Plotting 2D cuts.
  • Plotting phase space distributions.
Note

This package is still young, so be careful for any future breaking changes!

Warning

This package mostly aims at supporting Vlasiator 5.0+. Older versions of Vlasiator has different naming standard for outputs, and is not guaranteed to work.

Getting started

To install it,

pkg> add Vlasiator

You can then get started with

julia> using Vlasiator

More usages can be found in the examples.
